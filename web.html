Web Development Internship

Prerequisites

Participants from the following backgrounds are eligible to join this program:

  • CSE / IT
  • MCA / BCA / BSc IT
  • ECE

This program is ideal for:

  • Students preparing for placement drives
  • Professionals planning to switch to the Data Science domain

Infrastructural Requirements

Participants are expected to have the following:

  • Laptop with Microsoft Windows and minimum 8GB RAM
  • Laptop Charger / Adapter
  • Internet Connectivity
  • Smartphone (4G/5G)

Post Training Deliverables

Every participant will receive:

  • 50 hours training by a specialized trainer
  • Offer Letter
  • Certificate of Internship with Project Mentioned
  • Study Material / Classroom Material
  • Guidance for Entrepreneurship or Employment Support
  • Opportunity to assist trainers at workshops held in IISc, IITs, NITs, etc.

courses & content

Week 1: HTML & CSS Fundamentals

  • HTML5 structure and elements
  • CSS basics: selectors, properties
  • Box model, Flexbox, Grid
  • Mini Project: Personal Portfolio Page

Week 2: Responsive Design & Advanced CSS

  • Media queries for responsive design
  • Transitions, animations
  • Bootstrap basics
  • Mini Project: Responsive Blog Page

Week 3: JavaScript Basics

  • Syntax, variables, data types
  • DOM manipulation
  • Functions and events
  • Mini Project: To-do App

Week 4: Advanced JavaScript & ES6+

  • Arrow functions, let/const
  • Array methods (map, filter, reduce)
  • Promises and Fetch API
  • Mini Project: Weather App with API

Week 5: Version Control & GitHub

  • Git basics (init, commit, push)
  • Branching and merging
  • GitHub workflow
  • Mini Project: Host Project on GitHub

Week 6: React.js Fundamentals

  • Intro to React & JSX
  • Components, Props, State
  • Mini Project: React Task Manager

Week 7: Routing & APIs in React

  • React Router DOM
  • useEffect Hook & API Calls
  • Mini Project: Movie Search App

Week 8: Final Project & Deployment

  • Final Project: Full Portfolio Site
  • Deploy with Netlify/Vercel
  • Prepare GitHub and Resume
  • Project Demo & Presentation

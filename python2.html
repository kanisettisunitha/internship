Python Internship Syllabus (8 Weeks)

Prerequisites

Participants from the following backgrounds are eligible to join this program:

  • CSE / IT
  • MCA / BCA / BSc IT
  • ECE

This program is ideal for:

  • Students preparing for placement drives
  • Professionals planning to switch to the Data Science domain

Infrastructural Requirements

Participants are expected to have the following:

  • Laptop with Microsoft Windows and minimum 8GB RAM
  • Laptop Charger / Adapter
  • Internet Connectivity
  • Smartphone (4G/5G)

Post Training Deliverables

Every participant will receive:

  • 50 hours training by a specialized trainer
  • Offer Letter
  • Certificate of Internship with Project Mentioned
  • Study Material / Classroom Material
  • Guidance for Entrepreneurship or Employment Support
  • Opportunity to assist trainers at workshops held in IISc, IITs, NITs, etc.

courses & content

Week 1: Python Basics

  • Python installation & setup
  • Data types, variables, operators
  • Input/output, basic syntax
  • Mini Project: Calculator

Week 2: Control Flow & Functions

  • if-else, for & while loops
  • Functions, arguments, return values
  • Recursion
  • Mini Project: Number Guessing Game

Week 3: Data Structures

  • Lists, tuples, sets, dictionaries
  • Comprehensions
  • Sorting, searching techniques
  • Mini Project: Student Record Manager

Week 4: OOP in Python

  • Classes, objects
  • Inheritance, encapsulation, polymorphism
  • Dunder methods
  • Mini Project: Bank Account System

Week 5: File Handling & Exception Handling

  • Read/write files (.txt, .csv)
  • Try-except blocks, custom exceptions
  • With statements
  • Mini Project: Log File Analyzer

Week 6: Modules & Libraries

  • Standard modules: math, random, datetime
  • Installing libraries with pip
  • Working with requests, os, json
  • Mini Project: Weather App (API)

Week 7: Data & Visualization (Intro)

  • NumPy basics
  • Pandas DataFrames
  • Matplotlib/Seaborn plots
  • Mini Project: COVID Data Dashboard

Week 8: Final Project & Deployment

  • Capstone Project: Choose your own (CLI tool, dashboard, API)
  • Code review & optimization
  • Publish to GitHub
  • Presentation

Java Development Internship

Prerequisites

Participants from the following backgrounds are eligible to join this program:

  • CSE / IT
  • MCA / BCA / BSc IT
  • ECE

This program is ideal for:

  • Students preparing for placement drives
  • Professionals planning to switch to the Data Science domain

Infrastructural Requirements

Participants are expected to have the following:

  • Laptop with Microsoft Windows and minimum 8GB RAM
  • Laptop Charger / Adapter
  • Internet Connectivity
  • Smartphone (4G/5G)

Post Training Deliverables

Every participant will receive:

  • 50 hours training by a specialized trainer
  • Offer Letter
  • Certificate of Internship with Project Mentioned
  • Study Material / Classroom Material
  • Guidance for Entrepreneurship or Employment Support
  • Opportunity to assist trainers at workshops held in IISc, IITs, NITs, etc.

courses & content

Week 1: Java Basics & Setup

  • Installing JDK and IDE (Eclipse/IntelliJ)
  • Java syntax, data types, variables
  • Operators, input/output
  • Control structures: if, switch, loops
  • Mini Project: Simple Calculator

Week 2: Arrays, Strings & Methods

  • Single/Multi-dimensional Arrays
  • String handling and methods
  • Method overloading and parameters
  • Mini Project: Grade Analyzer

Week 3: Object-Oriented Programming (OOP)

  • Classes and Objects
  • Constructors, Encapsulation
  • Inheritance, Polymorphism
  • Abstract Classes & Interfaces
  • Mini Project: Bank Account Management

Week 4: Collections & Exception Handling

  • ArrayList, LinkedList, HashMap, HashSet
  • Iterators and enhanced for-loop
  • Try-catch, finally, custom exceptions
  • Mini Project: Student Record System

Week 5: File Handling & Streams

  • Reading and writing files
  • BufferedReader, FileWriter
  • Serialization & Deserialization
  • Mini Project: File Logger

Week 6: GUI with Swing/JavaFX

  • JFrame, JPanel, Layouts
  • JButton, JTextField, JLabel
  • Event handling
  • Mini Project: GUI Calculator

Week 7: JDBC & Database Integration

  • Connecting Java to MySQL
  • JDBC Driver and setup
  • CRUD operations using SQL
  • Mini Project: Inventory Manager

Week 8: Final Project & Deployment

  • Combine GUI + Database
  • Build and package JAR file
  • Deploy to GitHub
  • Final Project Demo

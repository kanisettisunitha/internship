App Development Internship

Prerequisites

Participants from the following backgrounds are eligible to join this program:

  • CSE / IT
  • MCA / BCA / BSc IT
  • ECE

This program is ideal for:

  • Students preparing for placement drives
  • Professionals planning to switch to the Data Science domain

Infrastructural Requirements

Participants are expected to have the following:

  • Laptop with Microsoft Windows and minimum 8GB RAM
  • Laptop Charger / Adapter
  • Internet Connectivity
  • Smartphone (4G/5G)

Post Training Deliverables

Every participant will receive:

  • 50 hours training by a specialized trainer
  • Offer Letter
  • Certificate of Internship with Project Mentioned
  • Study Material / Classroom Material
  • Guidance for Entrepreneurship or Employment Support
  • Opportunity to assist trainers at workshops held in IISc, IITs, NITs, etc.

courses & content

Week 1: Android & Java/Kotlin Basics

  • Install Android Studio
  • Overview of Android Architecture
  • Java/Kotlin basics for Android
  • Mini Project: Hello World App

Week 2: UI Components & Layouts

  • Linear, Relative, Constraint Layouts
  • TextView, Button, EditText, ImageView
  • Event handling and listeners
  • Mini Project: Basic Form App

Week 3: Activities & Intents

  • Activity Lifecycle
  • Explicit & Implicit Intents
  • Passing data between Activities
  • Mini Project: Multi-Screen Quiz App

Week 4: RecyclerView & Lists

  • RecyclerView Adapter & ViewHolder
  • CardView & List Optimization
  • OnClickListener for List Items
  • Mini Project: Contacts or Notes App

Week 5: Data Storage & SharedPreferences

  • Storing data locally
  • Using SharedPreferences
  • Introduction to SQLite/Room DB
  • Mini Project: Login System with Remember Me

Week 6: Working with APIs

  • Using Retrofit/Volley
  • Fetching data from REST APIs
  • Parsing JSON
  • Mini Project: Weather App or News Feed

Week 7: Advanced Features

  • Notifications
  • Background Tasks (WorkManager)
  • Permissions & Location
  • Mini Project: Alarm or Map App

Week 8: Final Project & Deployment

  • Final Full App (CRUD + API + UI)
  • Testing & Debugging
  • Publish to GitHub and Play Store (Optional)
  • Project Demo & Presentation
